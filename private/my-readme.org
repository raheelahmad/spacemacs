* Windows
** dependencies
- Ubuntu Fonts http://font.ubuntu.com/
- emacs-version ~25.0.92.1~

** Installation instructions
- clone repo as =~/.emacs.d=
- symlink =~/.emacs.d/.spacemacs= as =~/.spacemacs=

Environment variables
| name | value         |
|------+---------------|
| HOME | %userprofile% |

Things on ~PATH~

| name      | value                        |
|-----------+------------------------------|
| git tools | C:\Program Files\Git\usr\bin |
